A new application to simplify the process of manual prescription and decreasing the burdern of users to identify the pharmacy easily.Yes!!!
MediTrap allows doctors to send prescription to the patients digitally. It also provides space for patients and caretakers to identify the nearby pharmacies via google map. It supports three types of users such as doctors, patients or caretakers and admin. Registeration and login credentials are mandatory for all types of users. User interface changes according to the users. Doctors enter the prescription digitally and send to patients account via data stored in hospital database. User can view the digital prescription and also has the option to find the nearby pharmacies. If patient/caretaker is unable to identify the pharmacy where the prescribed drug is available, they can search through the search box. The google map will direct the user to the nearby pharmacy where the drug will be available.
